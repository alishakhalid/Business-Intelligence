**How to Run the Code (Google Colab)**
Follow these steps to run the notebook successfully:

1. Upload the data files to Google Drive
   Make sure the following files are saved in your Google Drive:
   - consumer_behavior_and_shopping_habits.csv
   - GDP_2023.csv

2. Update the file paths
   In the notebook, update the file paths so they point to the folder in your Google Drive where the CSV files are stored.

3. Open the notebook in Google Colab
   Go to: https://colab.google/
   Upload the notebook file or open it directly from your Google Drive.

4. Run the notebook
   Execute the cells in order.
